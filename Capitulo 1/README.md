# Abriendo el apetito
Este texto sirve como introducción motivacional al lenguaje de programación Python. Explica que Python es ideal para automatizar tareas repetitivas, desarrollar prototipos rápidos, crear aplicaciones con interfaz gráfica, juegos, bases de datos pequeñas o herramientas de prueba, especialmente cuando se quiere evitar la lentitud del ciclo de compilación típico en lenguajes como C, C++ o Java.

Se destaca que Python combina la facilidad de uso de los scripts de shell con la potencia de un lenguaje de programación completo, ofreciendo estructura para proyectos grandes, tipos de datos de alto nivel (como listas y diccionarios), y una sintaxis clara y concisa que reduce la cantidad de código necesario en comparación con otros lenguajes.

Además, Python es interpretado, lo que permite una programación interactiva y rápida iteración durante el desarrollo. Es modular, con una amplia biblioteca estándar, y extensible mediante código en C si es necesario. También se menciona que su nombre proviene del grupo de comedia británico Monty Python, no de la serpiente.

Finalmente, se invita al lector a comenzar a usar el intérprete de Python mientras sigue el tutorial, que progresará desde conceptos básicos (expresiones, tipos de datos, instrucciones) hasta temas avanzados (funciones, módulos, excepciones y programación orientada a objetos).
